# Phaser Vite Workspace Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the initial pnpm workspace and Phaser/Vite game app scaffold approved in the design spec.

**Architecture:** The repository root owns workspace metadata and shared tooling. `apps/game` is the only app package and contains a Phaser-only browser game module under `src/game`. The source boundary keeps Phaser logic independent from framework-specific UI so React can wrap the game later without moving core game files.

**Tech Stack:** Node 24, fnm, pnpm workspaces, Vite, Phaser, TypeScript, Vitest, jsdom, oxlint, oxfmt.

Source-level design guidance for the game app.

Create `apps/game/src/game/config.ts`:

```ts
import Phaser from "phaser";

import { BootScene } from "./scenes/BootScene";
import { GameScene } from "./scenes/GameScene";
import { PreloaderScene } from "./scenes/PreloaderScene";

export const GAME_PARENT_ID = "game-container";
export const GAME_WIDTH = 1280;
export const GAME_HEIGHT = 720;

export function createGameConfig(
  parent: string | HTMLElement = GAME_PARENT_ID,
): Phaser.Types.Core.GameConfig {
  return {
    type: Phaser.AUTO,
    parent,
    width: GAME_WIDTH,
    height: GAME_HEIGHT,
    backgroundColor: "#1f2933",
    scale: {
      mode: Phaser.Scale.FIT,
      autoCenter: Phaser.Scale.CENTER_BOTH,
    },
    physics: {
      default: "arcade",
      arcade: {
        gravity: { x: 0, y: 0 },
        debug: import.meta.env.DEV,
      },
    },
    scene: [BootScene, PreloaderScene, GameScene],
  };
}
```

Create `apps/game/src/game/createGame.ts`:

```ts
import Phaser from "phaser";

import { createGameConfig, GAME_PARENT_ID } from "./config";

export function getGameContainer(containerId = GAME_PARENT_ID): HTMLElement {
  const container = document.getElementById(containerId);

  if (!container) {
    throw new Error(`Missing game container: #${containerId}`);
  }

  return container;
}

export function createGame(containerId = GAME_PARENT_ID): Phaser.Game {
  const container = getGameContainer(containerId);

  return new Phaser.Game(createGameConfig(container));
}
```
